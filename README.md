# SphereNeRF

### Setup env
Create env using venv:
```bash
$ python3.9 -m venv .venv
```
or using conda:
```bash
$ conda create -n .venv python=3.9.9
```
Before requirements intallation:
```bash
$ sh env_setup.sh
```

install requirements:
```bash
$ source .venv/bin/activate
$ pip install -r requirements.txt
```
